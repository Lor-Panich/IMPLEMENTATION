# Product Report Image Builder

Local prototype สำหรับสร้างภาพรายงานสินค้าเป็น PNG ขนาด 1080 x 1080 px ด้วย HTML, CSS และ Vanilla JavaScript เท่านั้น

## วิธีเปิดใช้งาน local

1. เปิดไฟล์ `product-report-image-builder/index.html` ด้วย browser
2. ไม่ต้องติดตั้ง dependency
3. ไม่ต้องรัน backend หรือ dev server

## วิธีทดสอบ

1. เปิด `index.html`
2. อัปโหลดรูปสินค้า
3. เลือก layout รูปแบบ 1 รูป, 2 รูป หรือ 3 รูป
4. กรอกรหัสสินค้า เช่น `NC292`
5. กรอกรายละเอียดบรรทัดที่ 1 และ 2
6. `แม่แบบสถานะ` จะเริ่มที่ `กำหนดเอง` ให้พิมพ์ข้อความสถานะเอง
7. ลองบันทึก custom status preset แล้ว reload หน้าเพื่อตรวจว่ายังอยู่
8. ลองลบ custom status preset
9. กดเปิด `ตั้งค่าสีตัวอักษร` แล้วเลือกช่องสีที่ต้องการปรับ จากนั้นใช้ชุดสีด่วนหรือ color picker
10. กดเปิด `ตั้งค่าวันที่` ถ้าต้องการแก้วันที่เองหรือใช้ปุ่มวันที่วันนี้
11. กด `แสดงตัวอย่าง`
12. กด `ส่งออก PNG` แล้วตรวจว่าไฟล์ถูกดาวน์โหลด
13. กด `ล้างค่า` แล้วตรวจว่าฟอร์มและภาพกลับค่าเริ่มต้น แต่ custom presets ยังอยู่
14. ลองกรอกภาษาไทยและข้อความยาวเพื่อตรวจการตัดบรรทัด

## รายการฟีเจอร์

- อัปโหลดรูปสินค้า
- เลือก layout รูป 1 รูป, 2 รูป หรือ 3 รูป
- Preview อัตโนมัติหลังอัปโหลดรูป
- Canvas ขนาด 1080 x 1080 px
- รูปสินค้าอยู่ด้านบนและ fit โดยไม่บิดสัดส่วน
- รองรับรหัสสินค้า รายละเอียด 2 บรรทัด ข้อความสถานะ และวันที่
- แม่แบบสถานะเริ่มต้นมี `กำหนดเอง` เท่านั้น
- บันทึก custom status preset ไว้ใน localStorage และลบข้อความที่ผู้ใช้บันทึกเองได้
- เลือกสีข้อความหลัก สีขอบข้อความหลัก สีข้อความสถานะ และสีขอบข้อความสถานะ
- เลือกสีจาก panel สีแบบย่อหรือ color picker
- UI ภาษาไทยพร้อม section/card ที่อ่านง่ายขึ้น
- Dark mode สำหรับหน้าระบบ โดย canvas export ยังเป็นพื้นหลังสีขาว
- พับส่วนตั้งค่าสีตัวอักษรและตั้งค่าวันที่ไว้ก่อนเพื่อให้หน้าจอสั้นลง
- ปุ่มใช้วันที่วันนี้แบบไทย
- ข้อความหลักสีขาวพร้อมขอบดำ
- ข้อความสถานะสีแดงพร้อมขอบดำ
- ป้าย LIVE สีแดงและวันที่อยู่ด้านล่างขวา
- Export เป็น PNG
- ตั้งชื่อไฟล์ตาม `report-[รหัสสินค้า]-[วันที่].png`
- ถ้าไม่มีรหัสสินค้า ใช้ชื่อ `report-product.png`
- ถ้าไม่มีรูปในช่องใด แสดงพื้นที่ว่างพร้อมข้อความ `อัปโหลดรูป`
- ถ้าวันที่ว่าง ใช้วันที่วันนี้แบบไทยอัตโนมัติ
- ตัดบรรทัดข้อความยาวอัตโนมัติเพื่อไม่ให้ล้น Canvas

## สิ่งที่ยังไม่ได้ทำใน STEP 3C

- ยังไม่ต่อ Google Apps Script
- ยังไม่ต่อ Google Sheet
- ยังไม่ทำ Login
- ยังไม่บันทึกประวัติ
- ยังไม่มี backend
- ยังไม่มี production deployment
